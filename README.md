## Running Locally

Clone the project

```bash
git clone https://github.com/aditya-xq/terminal-portfolio.git
```

Go to the project directory

```bash
cd terminal-portfolio
```

Install dependencies

```bash
bun install
```

Start the server

```bash
bun run dev
```
